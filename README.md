**Modern, Fast, Clean.**
下一代高性能代理网关管理面板，旨在替代 Xboard/V2board。

## 🌟 核心特性
- 🚀 **双库支持**：完美兼容 **MySQL 8.0+** (虚拟主机) 和 **PostgreSQL 12+** (高性能 VPS)。
- 🎨 **Filament Admin**：基于 FilamentPHP 3.x 构建的现代化、零 Bug 管理后台。
- ⚡ **极速性能**：强制 Redis 队列，全异步架构，告别卡顿。
- 🛡️ **Gate 原生集成**：内置标准 UniProxy API，节点管理更稳定。
- 💻 **开源协议**：Apache 2.0，支持二开。
